Version 1.3.2

First new stable release since 1.3.0. Includes various improvements and bug fixes.
A few highlights:
  • New color way!
  • Image features: sigmas can be specified, 2D/3D filter selection
  • Training data no longer required for headless processing
  • Carving improvements: block-wise preprocessing
  • Various improvements and fixes to the object classification workflow
  • Two labels always added in Pixel and Object Classification
  • Fixed Fiji-MaMuT export plugin
For more details, see the Release Notes.

Previous stable versions

Version 1.3.0

We have updated all our dependencies, and switched from Python 2 to Python 3.
This could mean that projects created with older ilastik versions do not load any more and vice versa. We advise you to make copies of important project files before you try to use them with this new version of ilastik!
For more details, see the Release Notes.

Previous versions

ilastik 0.5 is still available here, along with its documentation. However, we would really encourage you to try a version > 1.0. If you rely on some functionality, which we did not yet port from ilastik 0.5, please contact us, and we'll try to find a workaround and re-adjust our priority list.
