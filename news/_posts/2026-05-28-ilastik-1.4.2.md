---
title: "New stable ilastik release 1.4.2"
---

We are happy to announce the release of the new stable [ilastik version `1.4.2`]({{site.baseurl}}/download.html#stable).

Among other improvements, the most noticeable change for many of you will be the new [Label Explorer]({{site.baseurl}}/documentation/basics/navigation#label-explorer).
Windows users (particularly on high-resolution screens) may notice that we switched to "High-DPI" mode, giving the interface a much more consistent look.

**Release Highlights:**
  * Physical pixel size metadata is now preserved when reading/exporting TIFF, H5 or OME-Zarr
  * Viewer performance upgrade: Browsing big datasets should feel much smoother now
  * New ["multi-scale OME-Zarr" export format]({{site.baseurl}}/documentation/basics/export#multiscale) and better UX when adding multiscale datasets to a project
  * New variant of Object Classification workflow ("Input: Label image"), which preserves object identity when using an instance segmentation from another tool such as Cellpose
